# Installation

Install the `@retikz/core` package using the following command (you can also use pnpm/yarn):

```bash
npm install @retikz/core
```

`@retikz/core` has a peer dependency: `react`. Please ensure that the corresponding package is already installed in your project:

```bash
npm install react react-dom
```

<Alert type="warn">Currently, @retikz/core is developed based on React 19 and will be compatible with React 18 before the official release. Support for lower versions is not being considered at this time.</Alert>
