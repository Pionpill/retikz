# 安装

通过如下指令安装 `@retikz/core` 包(也可以使用 pnpm/yarn):

```bash
npm install @retikz/core
```

你还需要确保安装了前置依赖项。你可能已经在项目中安装了 `react` 和 `react-dom`，但如果没有，请确保安装它们：

```bash
npm install react react-dom
```

<Alert title="兼容性" type="warning">最低兼容的 React 版本是 React18，且没有更低的版本兼容计划。</Alert>

## 包

我们开发 `retikz` 的最终目标是打造一个开发者友好的逻辑绘图工具，目前只有 `@retikz/core` 这一个包，包含了最基本的原子绘图语法，未来可能会发布 `@retikz/uml`，`retikz/plot` 等包。
