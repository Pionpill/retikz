<Update label="May 2026" tags={["@retikz/core", "@retikz/react", "docs"]}>

## v0.1

retikz redesign with IR at the center. Currently iterating on the `next` branch; merging into `main` ships it.

### `@retikz/core`

- **IR + scene compiler**: framework-agnostic intermediate representation and compiler, zero React and zero DOM
- **Persistence**: internal React state → serializable IR JSON, cross-platform and directly readable / writable by LLMs

### `@retikz/react`

- **Architecture split**: moved from a single React component library to a thin React adapter; core is decoupled from React
- **Kernel + Sugar JSX layers**: Kernel maps 1:1 to IR nodes; Sugar compiles down to Kernel
- peerDep React 18+

### `docs`

- v0.1 docs site: bilingual (zh / en) + co-located demos + live IR view

</Update>

<Update label="Apr 2025" tags={["retikz"]}>

## v0.0

First public series, iterating from `rc.0` through `rc.3`. This series shipped a single React package — no IR, no cross-framework architecture.

**Added**
- Arrow type: `Circle`
- Vertical way types: `|-|`, `-|-`

**Changed**
- React 18 compatibility

**Fixed**
- Fixed converted way bug
- Fixed `ScopeContext` propagation
- Various TypeScript errors and warnings
- Non-functional prop forwarding regressions

**Removed**
- `react/jsx-runtime` dependency dropped from build output

</Update>
