<Update label="2026.05" tags={["@retikz/core", "@retikz/react", "docs"]}>

## v0.1

retikz 重构，IR 居中。当前在 `next` 分支迭代，进入 `main` 即正式发布。

### `@retikz/core`

- **IR + Scene 编译器**：framework-agnostic 的中间表示与编译器，零 React、零 DOM
- **持久化形式**：内部 React state → 可序列化 IR JSON，跨平台、可被 LLM 直接读写

### `@retikz/react`

- **架构拆分**：从单包 React 组件库改为薄 React adapter，core 与 React 解耦
- **Kernel + Sugar 双层 JSX**：Kernel 直对应 IR 节点，Sugar 编译期展开为 Kernel
- peerDep React 18+

### `docs`

- v0.1 文档站：双语（zh / en）+ 同源 demo + IR 视图

</Update>

<Update label="2025.04" tags={["retikz"]}>

## v0.0

第一个公开版本系列，从 `rc.0` 迭代到 `rc.3`。该系列只发了 React 单包，没有 IR 与跨框架架构。

**Added**
- 箭头类型：`Circle`
- 竖向 way 类型：`|-|`、`-|-`

**Changed**
- 兼容 React 18

**Fixed**
- 修复 way 转换 bug
- 修复 `ScopeContext` 透传问题
- 修复若干 TypeScript 错误与告警
- 修复个别非功能性参数透传错误

**Removed**
- 构建产物移除对 `react/jsx-runtime` 的依赖

</Update>
