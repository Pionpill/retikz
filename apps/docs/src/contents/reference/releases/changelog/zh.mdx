## v0.1 — 开发中

retikz 重构，IR 居中。当前在 `next` 分支迭代，进入 `main` 即正式发布。

**Added**
- `@retikz/core`：IR + Scene 编译器（零 React、零 DOM）
- `@retikz/react`：React adapter，Kernel + Sugar 双层 JSX
- 文档站 v0.1，双语 + 同源 demo + IR 视图

**Changed**
- 架构由"单包 React 组件库"改为"core + 各框架薄 adapter"
- 持久化形式由内部 React state 改为可序列化 IR JSON

## v0.0 — 2025.01 → 2025.04

第一个公开版本系列，从 `rc.0` 迭代到 `rc.3`。该系列只发了 React 单包，没有 IR 与跨框架架构。

**Added**
- 箭头类型：`Circle`
- 竖向 way 类型：`|-|`、`-|-`

**Changed**
- 兼容 React 18

**Fixed**
- 修复 way 转换 bug
- 修复 `ScopeContext` 透传问题
- 修复若干 TypeScript 错误与告警
- 修复个别非功能性参数透传错误

**Removed**
- 构建产物移除对 `react/jsx-runtime` 的依赖
