## Current repository versions

| Package | Version | Notes |
| --- | --- | --- |
| `@retikz/core` | `0.3.0-beta.1` | IR schemas, geometry, parsers, and Scene compiler; zero React / zero DOM |
| `@retikz/render` | `0.3.0-beta.1` | SVG descriptors/strings, Canvas 2D, hydration helpers, and animation runtime utilities |
| `@retikz/react` | `0.3.0-beta.1` | React adapter with SVG / Canvas renderers |
| `@retikz/vanilla` | `0.3.0-beta.1` | Framework-free builder, SVG / Canvas mounting, hydration, animation controls, and SVG string output |

`0.3.0-beta.1` is the v0.3 pre-rc hardening window: the main renderer, hydration, Canvas, vanilla, and animation capabilities have landed, while the current work focuses on parity fixes, docs/package surface cleanup, optional export entries, and engineering cleanup before rc. See [Versioning](/about/releases/versioning).

## Feedback

retikz is in the v0.3 beta phase. Feedback should focus on SVG / Canvas parity, hydration and animation behavior, docs accuracy, installation issues, unreproducible examples, and real-project integration bugs before the rc freeze.
