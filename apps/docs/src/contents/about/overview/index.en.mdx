## Current repository versions

| Package | Version | Notes |
| --- | --- | --- |
| `@retikz/core` | `0.2.0-rc.1` | IR + scene compiler, zero React / zero DOM |
| `@retikz/react` | `0.2.0-rc.1` | React adapter + SVG renderer |

`0.2.0-rc.1` means v0.2 is in release-candidate validation: public APIs are frozen, and only compatible bug fixes, docs completion, installation acceptance, and release-note cleanup should land. `0.1.0` remains the previous stable line; `0.2.0` joins the stable / latest cadence only after the final cut. See [Versioning](/about/releases/versioning).

## Feedback

retikz is in the v0.2 rc phase. Feedback should focus on installation issues, missing docs, unreproducible examples, real-project integration issues, and bugs that can be fixed without breaking public APIs.
