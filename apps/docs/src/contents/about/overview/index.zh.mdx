## 当前仓库版本

| 包 | 版本 | 说明 |
| --- | --- | --- |
| `@retikz/core` | `0.2.0-rc.1` | IR + Scene 编译器，零 React / 零 DOM |
| `@retikz/react` | `0.2.0-rc.1` | React adapter + SVG renderer |

`0.2.0-rc.1` 表示 v0.2 已进入发布候选验收：公开 API 冻结，只接受兼容 bug fix、文档补齐、安装验收与发布说明收敛。`0.1.0` 仍是上一条稳定线；`0.2.0` 正式切出后才进入 stable / latest 节奏。详见[版本策略](/about/releases/versioning)。

## 反馈与提问

retikz 处于 v0.2 rc 阶段。反馈重点是安装问题、文档遗漏、示例不可复现、真实项目集成问题，以及不需要破坏公开 API 就能修复的 bug。
