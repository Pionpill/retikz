## 当前仓库版本

| 包 | 版本 | 说明 |
| --- | --- | --- |
| `@retikz/core` | `0.3.0-beta.1` | IR schema、几何工具、parser 与 Scene 编译器；零 React、零 DOM |
| `@retikz/render` | `0.3.0-beta.1` | SVG descriptor / string、Canvas 2D、水合 helper 与动画 runtime 工具 |
| `@retikz/react` | `0.3.0-beta.1` | React adapter，支持 SVG / Canvas renderer |
| `@retikz/vanilla` | `0.3.0-beta.1` | 无框架 builder、SVG / Canvas mount、水合、动画控制与 SVG string 输出 |

`0.3.0-beta.1` 是 v0.3 进入 rc 前的体验加固窗口：renderer、hydration、Canvas、vanilla 与动画主能力已经落地，当前重点是 parity 修复、文档与包描述刷新、可选导出入口和工程清理。详见 [版本策略](/about/releases/versioning)。
## 反馈与提问

retikz 处于 v0.3 beta 阶段。反馈重点是 SVG / Canvas parity、水合与动画行为、文档准确性、安装问题、示例不可复现，以及 rc 冻结前真实项目集成中暴露的 bug。
