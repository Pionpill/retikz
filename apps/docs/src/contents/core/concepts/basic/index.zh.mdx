基础概念放在 Concepts 的第一组，回答“一个点在哪里、这句话如何被解释”。读完这里，再看图元模型和图元关系会更顺；想理解 DSL、IR、Scene 和 renderer 的分工，请看设计理念。

## 职责一览

| 章节 | 主题 | 适合什么时候读 |
| --- | --- | --- |
| 坐标系 | 笛卡尔一等公民、极坐标、user units、viewBox 与显示尺寸 | 想知道 `[x, y]` 到底落在哪，或其它坐标写法最终如何变成笛卡尔点 |
| 定位 | 笛卡尔、极坐标、相对定位、偏移、比例点和路径相对 target | 想把节点或辅助点放到正确位置，或让路径走到某个目标 |
