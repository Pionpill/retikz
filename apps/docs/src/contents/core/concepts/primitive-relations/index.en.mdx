Primitive relations answer "how do these objects find one another?" If you are drawing flowcharts, concept maps, relationship lines, or small auto-layouts, read Positioning first, then Anchors.

## What each page covers

| Page | Topic | Reach for it when |
| --- | --- | --- |
| Positioning | Cartesian, polar, relative placement, offsets, proportional points, and path-relative targets | You want to place nodes / helper points or send a path to a target |
| Anchors | Node boundaries, auto-attach, explicit anchors, edge-proportional points, and margin | You want to control which side a connection leaves from, where it lands, or how much space it keeps |
