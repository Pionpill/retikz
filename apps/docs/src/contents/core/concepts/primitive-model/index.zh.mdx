图元模型回答“图里有哪些对象，以及它们如何被渲染”。如果你想先理解 `<Node>`、`<Path>`、`<Coordinate>`、`<Scope>` 这些基础对象和渲染管线的关系，可以从这里开始。

## 职责一览

| 章节 | 主题 | 适合什么时候读 |
| --- | --- | --- |
| 分层 | DSL、IR、Scene 与渲染器之间的关系 | 想知道 JSX 怎么变成 SVG / Canvas，或想理解 Kernel / Sugar 的边界 |
| 动画 | 图元属性、路径绘制、镜头等随时间变化的规则 | 想让图元进入、闪烁、旋转、描边，或控制整张图的播放 |
