图元模型回答“一个图元自己由什么构成、具有什么行为”。这里先从最常用的 Node 模型讲起，再看图元属性如何随时间变化；图元如何进入渲染管线请看设计理念里的分层。

## 职责一览

| 章节 | 主题 | 适合什么时候读 |
| --- | --- | --- |
| Node 模型 | 内容盒、padding、shape、外接框、boundary 与 anchor 的分工 | 想理解节点为什么这么大、线为什么接到这里、形状如何参与布局 |
| 动画 | 图元属性、路径绘制、镜头等随时间变化的规则 | 想让图元进入、闪烁、旋转、描边，或控制整张图的播放 |
