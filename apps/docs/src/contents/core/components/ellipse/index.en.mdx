`<Ellipse>` draws an ellipse in one line: give a center + two radii (or two diameters / bounding-box corners); add angles for a partial ellipse. It expands to `<Path>` + `<Step kind="ellipsePath">`.

Field names follow the IR: `radiusX` / `radiusY` (not `xRadius` / `yRadius`). For a labelled, connectable ellipse node use [`<Node shape="ellipse">`](/core/components/node/overview).

## Usage

```tsx
import { Ellipse } from '@retikz/react';
```

```tsx
<Ellipse center={[0, 0]} radiusX={60} radiusY={40} />
```

## Forms

| Form | prop | Point type |
| --- | --- | --- |
| center + two radii | `{ center, radiusX, radiusY }` | `center` any Target |
| center + two diameters | `{ center, diameterX, diameterY }` | `center` any Target |
| bounding box (inscribed) | `{ corner1, corner2 }` | literal `[x, y]` |

## Partial cropping

Same as [Circle](/core/components/circle): any two of `startAngle` / `endAngle` / `sweepAngle` + `closed` (`chord` default / `open`). No angles = full ellipse.

## API reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `center` | `Target` | — | Center (radii / diameter forms) |
| `radiusX` / `radiusY` | `number` | — | The two semi-axes |
| `diameterX` / `diameterY` | `number` | — | The two diameters (= semi-axis × 2) |
| `corner1` / `corner2` | `[x, y]` | — | Inscribed-ellipse bounding box (literal) |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | Partial-ellipse angles (any two) |
| `closed` | `'chord' \| 'open'` | `'chord'` | Partial-ellipse closing mode |
| Visual props | same as `<Path>` | — | `fill` / `stroke` / `zIndex` etc. |
