`<Ellipse>` 用一行画椭圆：给中心 + 两半轴（或两直径 / 包围盒两角）；带角度时画部分椭圆。它编译期展开为 `<Path>` + `<Step kind="ellipsePath">`。

字段名沿用 IR：`radiusX` / `radiusY`（不是 `xRadius` / `yRadius`）。要画含文字、可连线的椭圆节点用 [`<Node shape="ellipse">`](/core/components/node/overview)。

## 用法

```tsx
import { Ellipse } from '@retikz/react';
```

```tsx
<Ellipse center={[0, 0]} radiusX={60} radiusY={40} />
```

## 形态

| 形态 | prop | 点位 |
| --- | --- | --- |
| 中心 + 两半轴 | `{ center, radiusX, radiusY }` | `center` 任意 Target |
| 中心 + 两直径 | `{ center, diameterX, diameterY }` | `center` 任意 Target |
| 包围盒两角（内切椭圆） | `{ corner1, corner2 }` | 限 literal `[x, y]` |

## 部分裁剪

同 [Circle](/core/components/circle)：`startAngle` / `endAngle` / `sweepAngle` 三选二 + `closed`（`chord` 默认 / `open`）。不给角度 = 整椭圆。

## API 参考

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `center` | `Target` | — | 中心（半轴 / 直径形态） |
| `radiusX` / `radiusY` | `number` | — | 两半轴 |
| `diameterX` / `diameterY` | `number` | — | 两直径（= 半轴 × 2） |
| `corner1` / `corner2` | `[x, y]` | — | 内切椭圆包围盒（literal） |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | 部分椭圆角度（三选二） |
| `closed` | `'chord' \| 'open'` | `'chord'` | 部分椭圆闭合模式 |
| 视觉 prop | 同 `<Path>` | — | `fill` / `stroke` / `zIndex` 等 |
