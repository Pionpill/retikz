`<Circle>` 用一行画圆：给圆心 + 半径（或直径 / 直径两端 / 包围盒两角）即可；带角度时画部分圆（半圆 / 弓形）。它编译期展开为 `<Path>` + `<Step kind="circlePath">`——遵循"Sugar 不引入新能力"。

与 `<Node shape="circle">` 的区别：`<Circle>` 是**无文本的纯绘制路径**；Node 圆是带文本 / 锚点、能被边连接的**对象**。要画含文字、可连线的圆节点用 [`<Node>`](/core/components/node/overview)。

## 用法

```tsx
import { Circle } from '@retikz/react';
```

```tsx
<Circle center={[0, 0]} radius={40} />
```

## 形态

圆心 + 半径四选一：

| 形态 | prop | 点位 |
| --- | --- | --- |
| 圆心 + 半径 | `{ center, radius }` | `center` 任意 Target |
| 圆心 + 直径 | `{ center, diameter }` | `center` 任意 Target |
| 直径两端 | `{ from, to }` | 限 literal `[x, y]` |
| 包围盒两角 | `{ corner1, corner2 }`（半径 = min(\|dx\|,\|dy\|)/2） | 限 literal `[x, y]` |

需在组件内算坐标的形态（`from/to`、`corner1/corner2`）点位**只接 literal 笛卡尔 `[x, y]`**，传 node id / 极坐标会报错。

## 部分裁剪

给 `startAngle` / `endAngle` / `sweepAngle` 中**任意两个**画部分圆：

- `closed="chord"`（默认）：弦闭合 → 半圆 / 弓形
- `closed="open"`：纯弧线（等价 [`<Arc>`](/core/components/arc)）

不给角度 = 整圆。

## API 参考

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `center` | `Target` | — | 圆心（`{center,radius}` / `{center,diameter}` 形态） |
| `radius` | `number` | — | 半径 |
| `diameter` | `number` | — | 直径（= radius × 2） |
| `from` / `to` | `[x, y]` | — | 直径两端（literal） |
| `corner1` / `corner2` | `[x, y]` | — | 包围盒两角（literal） |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | 部分圆角度（三选二） |
| `closed` | `'chord' \| 'open'` | `'chord'` | 部分圆闭合模式 |
| 视觉 prop | 同 `<Path>` | — | `fill` / `stroke` / `strokeWidth` / `dashPattern` / `opacity` / `zIndex` 等 |
