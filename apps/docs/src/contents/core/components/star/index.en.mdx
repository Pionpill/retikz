title: Star
description: Shape sugar — draw a star in one line; expands to Path (move + 2N line + cycle, alternating outer/inner radius).
---

`<Star>` draws a star: give a center + outer radius + number of points (inner radius given explicitly or by ratio). It is a **pure geometry sugar with no IR change** — it expands to a `<Path>`: `2·points` vertices alternating outer / inner radius + `cycle`.

## Usage

```tsx
import { Star } from '@retikz/react';
```

```tsx
<Star center={[0, 0]} outerRadius={40} points={5} fill="#fbbf24" />
```

5-point star (default inner ratio 0.5) + 6-point star (`innerRatio={0.6}`). One outer point faces up by default.

## Forms

| Form | prop |
| --- | --- |
| center + outer/inner radius + points | `{ center, outerRadius, innerRadius, points }` |
| center + outer radius + points (ratio inner) | `{ center, outerRadius, points, innerRatio? }` (default `innerRatio = 0.5`) |

`center` must be **literal `[x, y]`**. `points` is an integer `>= 2`.

## API reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `center` | `[x, y]` | — | Center (literal) |
| `outerRadius` | `number` | — | Outer radius (point vertices) |
| `innerRadius` | `number` | — | Inner radius (notch vertices; xor innerRatio) |
| `innerRatio` | `number` | `0.5` | Inner/outer radius ratio (`innerRadius = outerRadius × innerRatio`) |
| `points` | `number` (integer ≥ 2) | — | Number of points |
| `rotate` | `number` | `-90` | Overall start angle (°); one outer point faces up by default |
| Visual props | same as `<Path>` | — | `fill` / `stroke` / `fillRule` / `zIndex` etc. |
