---
title: 节点
description: 节点家族——可被引用的命名图元，连同它的行级文字样式与纯占位点。
---

节点是图里「被引用」的主体：它有 id、有位置、有形状边界，连线会按 id 自动贴上它的边界。这一分组把节点本身、写在节点内部的行级文字样式、以及只命名不画形状的占位点收在一起。

<ComponentPreview name="node-family" size="md" hideCode />

## 职责一览

| 组件 | 类型 | 职责 | 适合场景 |
| --- | --- | --- | --- |
| Node | Kernel | 命名图元，带位置 / 形状 / 文本，路径靠 id 贴它的边界 | 流程图节点、带文字的形状、任何要被连线引用的图元 |
| Text | Sugar | 写在 Node 内部，给单独一行覆盖 fill / opacity / font | 一行里混排多种样式的文字 |
| Coordinate | Kernel | 只命名一个点，不画任何形状 | 纯引用锚点、辅助定位、不需要可见图形时 |
