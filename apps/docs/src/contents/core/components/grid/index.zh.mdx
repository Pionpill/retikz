`<Grid>` 画网格底纹：给包围盒（两角或中心 + 尺寸）+ 步长，展开为**多条** `<Path>`（竖线 + 横线，每条 move + line，互不闭合）。视觉 prop 透传到每条线。

常配 `zIndex`（设负值沉到底）或放在内容之前作底纹。点位限 literal 笛卡尔（要算每条线端点）。

## 用法

```tsx
import { Grid } from '@retikz/react';
```

```tsx
<Grid corner1={[0, 0]} corner2={[100, 60]} step={10} stroke="#ddd" />
```

## 形态与步长

| 形态 | prop |
| --- | --- |
| 两角 | `{ corner1, corner2 }` |
| 中心 + 尺寸 | `{ center, width, height }` |

步长（点位都限 literal `[x, y]`）：

- `step`：单值，xStep = yStep
- `xStep` + `yStep`：分轴（覆盖 `step`）

**边界规则**：按 `corner1`（或 `center - 尺寸/2`）起算，严格按 step 步进；最后一条若与对角不重合就不画（不补尾）。竖线 `floor(|x1-x0|/xStep)+1` 条，横线同理。

## API 参考

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `corner1` / `corner2` | `[x, y]` | — | 包围盒两角（literal） |
| `center` | `[x, y]` | — | 中心（中心形态，literal） |
| `width` / `height` | `number` | — | 尺寸（中心形态） |
| `step` | `number` | — | 单值步长 |
| `xStep` / `yStep` | `number` | — | 分轴步长（覆盖 step） |
| 视觉 prop | 同 `<Path>` | — | `stroke` / `strokeWidth` / `dashPattern` / `zIndex` 等 |
