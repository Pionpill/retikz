---
title: Shapes
description: 用更短的 JSX 写法表达常见图形，底层仍会展开成 Kernel 组件。
---

这里展示的内容，都是把常见绘图意图写短：比如一行画圆、一行画网格、用数组描述一条连线。它们在实现上属于一类便捷写法，会在 React 包里展开成 `<Path>`、`<Step>`、`<Draw>` 这类 Kernel / Sugar 组合。

`<Draw>` 也属于这类便捷写法，但它太常用了，所以单独放在 [绘制](/core/components/draw/overview) 分组里，不在这里重复展开。

## 组成

| 页面 | 类型 | 职责 | 适合场景 |
| --- | --- | --- | --- |
| Circle / Ellipse | Shapes | 圆、椭圆、半圆和弓形 | 想从中心、半径、直径或包围盒快速画圆形 |
| Arc / Sector | Shapes | 弧线、扇形和空心扇形 | 想表达角度范围、扇区高亮或环形片段 |
| Rectangle / Polygon | Shapes | 矩形、圆角矩形和规则多边形 | 想画四边形或等边多边形，不想手写顶点 |
| Star | Shapes | 内外半径交替的星形 | 想画评分星、强调标记或装饰性多角形 |
| Grid | Shapes | 网格、辅助线、边框、主次线 | 想画一张网格或一组规则参考线 |
