星形在外半径和内半径之间交替取点，retikz 里有两种造法：

- **画一条轮廓**——Sugar 组件 `<Star>` 在指定位置画一个星形，展开成底层 `<Path>`。
- **一个带边界的节点**——`<Node shape={{ type:'star', params }}>` 让节点以星形为边界，可被连线引用、当作评分 / 强调 glyph。

简单说：只画一个星形，用 `<Star>`；要一个可连接的星形图元，用 `<Node shape>`。

## 用法

```tsx
import { Star, Node } from '@retikz/react';
```

```tsx
{/* 画一条轮廓 */}
<Star center={[0, 0]} outerRadius={36} innerRadius={16} points={5} />

{/* 一个带星形边界的节点 */}
<Node shape={{ type: 'star', params: { points: 5, innerRadius: 16, outerRadius: 40 } }} />
```

## 例子

### 作为图形：Star

`points` 是星角数量。你可以显式给 `innerRadius`，也可以用 `innerRatio` 按比例自动算内半径。

| 写法 | 适合 |
| --- | --- |
| `{ center, outerRadius, innerRadius, points }` | 自己控制内外半径 |
| `{ center, outerRadius, points, innerRatio? }` | 用比例自动计算内半径 |

<ComponentPreview name="star-basic" size="sm" />

### 作为节点形状：shape="star"

当你要的是一个星形**节点**（可被连线引用、当图标 glyph）而不是一条线时，给 `<Node>` 设 `shape={{ type:'star', params }}`。尺寸由半径参数定、**不依赖文字内容**。

| shape | params | 说明 |
| --- | --- | --- |
| star | `{ points, innerRadius, outerRadius, rotate? }` | 内外半径交替的星形 |

<ComponentPreview name="node-star" size="sm" />

除节点的 9 个标准 anchor（`north` / `center` …）外，星形还提供按几何命名的 anchor，可用 `{ id, anchor }` 精确连线：`tip-N`（第 N 个尖角）/ `notch-N`（第 N 个凹角）/ `center`。完整的 `<Node>` 属性见 [Node](/core/components/node/overview)。

## 技术原理

> 日常使用可以跳过本节；只有在星形朝向或内外半径比例不符合预期时再回来读。

`<Star>` 会先计算 `points * 2` 个顶点：外点落在 `outerRadius` 上，内点落在 `innerRadius` 上。默认 `innerRatio` 是 `0.5`，也就是内半径为外半径的一半。

`rotate` 改变第一个外点的起始角度。默认起始角让一个外点朝上；在屏幕坐标里，正角度会顺时针旋转。`<Node shape="star">` 与 `<Star>` 几何一致，区别只是前者是有边界、可连线的节点。

## API 参考

### 作为图形：Star

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `center` | `[number, number]` | — | 星形中心 |
| `outerRadius` | `number` | — | 外半径 |
| `innerRadius` | `number` | — | 内半径；显式给出时优先 |
| `innerRatio` | `number` | `0.5` | 未给 `innerRadius` 时，内半径相对外半径的比例 |
| `points` | `number` | — | 星角数量 |
| `rotate` | `number` | `-90` | 起始角度 |

### 作为节点形状

| `shape` | params | 边界 / 用途 |
| --- | --- | --- |
| `{ type: 'star', params }` | `points` / `innerRadius` / `outerRadius` / `rotate?` | 可连接的星形 glyph |

> `<Node>` 的完整属性见 [Node](/core/components/node/overview)。
