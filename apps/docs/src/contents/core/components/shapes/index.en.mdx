This page groups the shorthand forms that let you write common drawing intent in a shorter way: draw a circle in one line, draw a grid in one line, or describe a connection with an array. In the React package they still expand to Kernel / Sugar combinations such as `<Path>`, `<Step>`, and `<Draw>`.

`<Draw>` also belongs to this family of shortcut forms, but it is common enough to live in its own [Draw](/core/components/draw/overview) group, so we do not repeat it here.

## What it contains

| Page | Type | Responsibility | Use it when |
| --- | --- | --- | --- |
| Circle / Ellipse | Shapes | Circles, ellipses, half circles, and segments | You know a center, radius, diameter, or bounding box |
| Arc / Sector | Shapes | Arcs, sectors, and donut sectors | You want angle marks, sector highlights, or ring fragments |
| Rectangle / Polygon | Shapes | Rectangles, rounded rectangles, and regular polygons | You want boxes or equal-sided polygons without hand-writing vertices |
| Star | Shapes | Alternating inner / outer radius stars | You want rating stars, emphasis marks, or decorative multi-point shapes |
| Grid | Shapes | Grids, guide lines, borders, and major lines | You want a background grid or a set of regular reference lines |
