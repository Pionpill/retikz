`<Grid>` 是专门画规则参考线的语法糖。你给出范围和步长，它会展开成多条 `<Path>`：横线、竖线、可选主线和可选外边框都由底层路径绘制。

它适合做背景网格、坐标纸、辅助参考线，也可以只画一组横线或竖线。圆、矩形、星形这类单个几何形状放在 [Shapes](/core/components/shapes) 分组；`Grid` 因为有对齐、主次线和边框控制，所以单独成页。

## 用法

```tsx
import { Grid } from '@retikz/react';
```

```tsx
<Grid corner1={[20, 20]} corner2={[260, 150]} step={20} stroke="lightgray" />
```

### 什么时候用

| 场景 | 推荐写法 |
| --- | --- |
| 普通背景网格 | `corner1` + `corner2` + `step` |
| 已知中心和尺寸 | `center` + `width` + `height` + `step` |
| 横纵间距不同 | `xStep` + `yStep` |
| 只要横线或竖线 | `showHorizontal` / `showVertical` |
| 每隔几条线强调一次 | `majorEvery` + `majorStroke` / `majorStrokeWidth` |
| 网格外需要框线 | `border` + `border*` |

## 技术原理

> 日常使用可以跳过本节；只有在调试层级、边界线或主次线顺序时再回来读。

### Grid 展开为 Path

`<Grid>` 不会产生新的底层图元。它会按 x / y 方向计算所有需要出现的坐标，然后为每条线生成一条 `<Path>`。如果启用 `border`，外边框会额外生成一条闭合 `<Path>`。

这意味着普通路径视觉属性可以直接用于网格线，例如 `stroke`、`strokeWidth`、`dashPattern`、`opacity` 和 `zIndex`。主线用 `major*` 覆盖普通线样式，外边框用 `border*` 覆盖普通线样式。

### 对齐从 origin 开始

默认情况下，网格从范围左上角开始排线。设置 `origin` 后，线会按这个基准点向两侧等距展开；`offset` 会在 `origin` 基础上再平移一个相位。

`includeBoundary` 不改变步长，只是在范围两端额外补边界线。范围不能被步长整除时，它能保证最外侧也有参考线。

### 外框不等于边界线

边界线仍然是网格线的一部分，样式跟普通线或主线走；外框是额外的一条闭合路径，可以拥有自己的 `borderStroke`、`borderStrokeWidth`、`borderDashPattern`、`borderPadding` 等样式。

`borderPadding` 会把外框向外扩出一圈。默认网格线仍停在原始范围内；如果希望网格线延伸到外框，设置 `clipToBorder`。

## API 参考

### 范围与步长

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `corner1` + `corner2` | `[number, number]` | — | 用两个对角点确定网格范围 |
| `center` + `width` + `height` | `[number, number]` + `number` | — | 用中心和尺寸确定网格范围 |
| `step` | `number` | — | 横纵方向共用的步长 |
| `xStep` | `number` | — | x 方向步长；会覆盖 `step` |
| `yStep` | `number` | — | y 方向步长；会覆盖 `step` |
| `showVertical` | `boolean` | `true` | 是否绘制竖线 |
| `showHorizontal` | `boolean` | `true` | 是否绘制横线 |
| `origin` | `[number, number]` | 范围左上角 | 网格对齐基准点 |
| `offset` | `[number, number]` | `[0, 0]` | 在 `origin` 基础上的相位偏移 |
| `includeBoundary` | `boolean` | `false` | 范围不能被步长整除时，是否补上边界线 |

### 主线

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `majorEvery` | `number` | — | 每隔几条线使用主线样式 |
| `majorOffset` | `number` | `0` | 主线计数偏移 |
| `majorColor` | `string` | — | 主线 `color` |
| `majorStroke` | `string` | — | 主线描边 |
| `majorStrokeWidth` | `number` | — | 主线描边宽度 |
| `majorDashPattern` | `Array<number>` | — | 主线虚线段和间隔 |
| `majorLineCap` | `'butt' \| 'round' \| 'square'` | — | 主线端点形状 |
| `majorLineJoin` | `'miter' \| 'round' \| 'bevel'` | — | 主线拐点形状 |
| `majorThickness` | `PathThickness` | — | 主线语义粗细 |
| `majorOpacity` | `number` | — | 主线整体透明度 |
| `majorDrawOpacity` | `number` | — | 主线描边透明度 |

### 外边框

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `border` | `boolean` | `false` | 是否绘制外边框 |
| `borderPadding` | `number` | `0` | 外边框与网格范围之间的距离 |
| `borderRenderOrder` | `'before' \| 'after'` | `'after'` | 外边框绘制在网格线之前还是之后 |
| `clipToBorder` | `boolean` | `false` | 网格线是否延伸到外边框 |
| `borderColor` | `string` | — | 外边框 `color` |
| `borderStroke` | `string` | — | 外边框描边 |
| `borderStrokeWidth` | `number` | — | 外边框描边宽度 |
| `borderDashPattern` | `Array<number>` | — | 外边框虚线段和间隔 |
| `borderLineCap` | `'butt' \| 'round' \| 'square'` | — | 外边框端点形状 |
| `borderLineJoin` | `'miter' \| 'round' \| 'bevel'` | — | 外边框拐点形状 |
| `borderThickness` | `PathThickness` | — | 外边框语义粗细 |
| `borderFill` | `string \| PaintSpec` | — | 外边框闭合区域填充 |
| `borderFillRule` | `'nonzero' \| 'evenodd'` | — | 外边框填充规则 |
| `borderOpacity` | `number` | — | 外边框整体透明度 |
| `borderFillOpacity` | `number` | — | 外边框填充透明度 |
| `borderDrawOpacity` | `number` | — | 外边框描边透明度 |
| `borderZIndex` | `number` | — | 外边框层级 |
