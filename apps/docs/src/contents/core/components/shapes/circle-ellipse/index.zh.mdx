`<Circle>` 和 `<Ellipse>` 都是把常见几何写法展开成 `<Path>` 的语法糖。它们适合那些“我知道中心和半径”“我只有一个包围盒”“我想在一块矩形区域里塞一个圆或椭圆”的场景。

## 用法

```tsx
import { Circle, Ellipse } from '@retikz/react';
```

```tsx
<Circle center={[0, 0]} radius={32} fill="none" />
<Ellipse center={[0, 0]} radiusX={48} radiusY={28} fill="none" />
```

## 示例

### Circle

最常见的是 `center + radius`。如果手上是一个矩形范围，也可以直接用 `corner1 + corner2`、`box` 或 `boundingBox`。

| 写法 | 适合 |
| --- | --- |
| `{ center, radius }` | 已知圆心和半径 |
| `{ center, diameter }` | 已知直径 |
| `{ from, to }` | 直径两端点 |
| `{ corner1, corner2 }` | 放进一个包围盒 |
| `{ box }` / `{ boundingBox }` | 用 `x/y/origin + width + height` 直接描述矩形范围 |
| + `fit="contain" \| "cover"` | 让圆按范围内切或外接 |
| + `inset` / `outset` | 在原范围基础上先收缩或扩展再绘制 |
| + angles + `closed="sector"` | 画 1/4 圆、30° 圆这类连回圆心的残缺圆 |

### Ellipse

椭圆和圆很像，只是半径拆成了 `radiusX` 和 `radiusY`。它也支持 `corner1 + corner2`、`box` 和 `boundingBox`。

| 写法 | 适合 |
| --- | --- |
| `{ center, radiusX, radiusY }` | 已知中心和两个半轴 |
| `{ center, diameterX, diameterY }` | 已知两个直径 |
| `{ corner1, corner2 }` | 放进一个包围盒 |
| `{ box }` / `{ boundingBox }` | 用 `x/y/origin + width + height` 直接描述矩形范围 |
| + `inset` / `outset` | 先收缩或扩展范围再画椭圆 |
| + angles + `closed="sector"` | 画连回中心的部分椭圆 |

## 技术原理

> 日常使用可以跳过本节；只有在点位报错，或者想把它们和 `<Path>` 混用时再回来读。

### 有些写法直接透传点，有些写法要先计算

有些写法是直接把点交给底层 `Path`。它们可以用节点 id、极坐标点或普通 `[x, y]` 坐标：

```tsx
<Circle center="A" radius={24} />
<Ellipse corner1="A" corner2="B" />
```

另外一些写法要先在 React 里算出圆心、尺寸或边界，所以只接受字面量 `[x, y]` 坐标：

```tsx
<Circle from={[-40, 0]} to={[40, 0]} />
```

把节点 id 传给这类需要先计算的写法时，组件会直接报错，不会替你猜结果。

### 部分圆有三种闭合方式

带角度时，`closed` 决定弧线两端怎么收口：

| 值 | 效果 |
| --- | --- |
| `chord` | 默认值，直接连接两个弧端点 |
| `sector` | 从弧终点连到中心，再回到弧起点 |
| `open` | 只画弧线，不闭合 |

### 它们最后还是 Path

`<Circle>` 和 `<Ellipse>` 最后都会展开成等价的 `<Path>` / `<Step>` 序列。这意味着它们可以直接用 `stroke`、`fill`、`arrow`、`zIndex` 这类 Path 风格属性。

## API 参考

| 组件 | 组合 | 含义 |
| --- | --- | --- |
| `<Circle>` | `center` + `radius` | 用圆心和半径画圆 |
| | `center` + `diameter` | 用圆心和直径画圆 |
| | `from` + `to` | 用两点画圆 |
| | `corner1` + `corner2` | 用包围盒画内切圆 |
| | `box` / `boundingBox` | 用矩形范围描述圆，配合 `fit` / `inset` / `outset` |
| | + `fit?` | `contain` 取内切，`cover` 取外接 |
| | + `inset?` / `outset?` | 收缩或扩展 box 范围后再计算 |
| | + angles (`startAngle` / `endAngle` / `sweepAngle`, any two) + `closed?` | 部分圆：`chord` / `sector` / `open` |
| `<Ellipse>` | `center` + `radiusX` + `radiusY` | 用中心和两个半轴画椭圆 |
| | `center` + `diameterX` + `diameterY` | 用中心和两个直径画椭圆 |
| | `corner1` + `corner2` | 用包围盒画内切椭圆 |
| | `box` / `boundingBox` | 用矩形范围描述椭圆，配合 `inset` / `outset` |
| | + `inset?` / `outset?` | 收缩或扩展 box 范围后再计算 |
| | + angles + `closed?` | 部分椭圆：`chord` / `sector` / `open` |
