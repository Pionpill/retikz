---
title: Shapes
description: 同一个几何形状的两种造法——用 Sugar 画一条轮廓，或让节点带上这个形状的边界。
---

这一组是常见的几何形状：圆、椭圆、弧、扇形、矩形、多边形、星形。retikz 里每种形状都有**两种造法**，每页把它们并列：

- **画一条轮廓**——用 Sugar 组件（`<Circle>` / `<Sector>` …）在指定位置画一条形状线，它会展开成底层 `<Path>`。纯图形，不参与连线。
- **一个带边界的节点**——用 `<Node shape="circle">`，得到一个以该形状为边界的节点：能装文字、能被 `<Path>` / `<Draw>` 用 id 引用、连线端点自动贴形状的边。

简单说：只要一条形状线，用 Sugar；要一个可连接、可标注的形状图元，用 Node 形状。

## 组成

| 形状 | 画一条轮廓（Sugar） | 带边界的节点（Node 形状） |
| --- | --- | --- |
| Circle / Ellipse | `<Circle>` · `<Ellipse>` | `shape="circle"` · `"ellipse"` |
| Arc / Sector | `<Arc>` · `<Sector>` | `shape="arc"` · `"sector"` |
| Rectangle | `<Rectangle>` | 默认 / `shape="rectangle"` |
| Polygon | `<RegularPolygon>` | `shape="polygon"` · `"diamond"` |
| Star | `<Star>` | `shape="star"` |

> 背景网格、参考线这类**一片规则线条**（而非单个闭合形状）不在这一组——见 [辅助](/core/components/helpers) 里的 `<Grid>`。
