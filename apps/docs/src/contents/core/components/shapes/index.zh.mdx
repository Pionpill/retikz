title: Shapes

---

形状（圆、椭圆、弧、扇形、矩形、多边形、星形）是 retikz 的一类几何原语，有**两副面孔**：

- **作为 Path 图形（Sugar）**——`<Circle>` `<Sector>` 这类组件是官方提供的一组形状快捷写法，在指定位置画一条形状线、展开成底层 `<Path>`。它们让你不必为常见图形手写 Path / Step。
- **作为 Node 形状**——`<Node shape="circle">` 让节点以该形状为边界：能装文字、能被 `<Path>` / `<Draw>` 用 id 引用、连线端点自动贴边。

每个形状页把这两副面孔分两大块写全：先「作为 Path 图形」，再「作为 Node 形状」。简单说：要画一条形状线，用 Sugar；要一个可连接、可标注的形状节点，用 `<Node shape>`。

**两侧都对扩展开放**：

- 想要内置之外的 **Node 形状**（hexagon / cloud…）——注册到 [shape-registry](/core/reference/extending/shape-registry)；注册表里的内置项是 rectangle / ellipse / sector / arc / polygon / star（circle / diamond 是编译期 preset 别名，不在注册表）。
- 想画内置之外的**路径曲线 step**（parabola / sin-cos…）——走 [path-generator](/core/reference/extending/path-generator)。

## 职责一览

| 形状 | 作为 Path（Sugar） | 作为 Node 形状 |
| --- | --- | --- |
| Circle / Ellipse | `<Circle>` · `<Ellipse>` | `shape="circle"` · `"ellipse"` |
| Arc / Sector | `<Arc>` · `<Sector>` | `shape="sector"` · `"arc"`（带 params） |
| Rectangle | `<Rectangle>` | 默认 / `shape="rectangle"` |
| Polygon | `<RegularPolygon>` | `shape={{ type:'polygon', params }}`（带 params）· `"diamond"` |
| Star | `<Star>` | `shape={{ type:'star', params }}`（带 params） |

> 背景网格、参考线这类**一片规则线条**（而非单个闭合形状）不在这一组——见 [辅助](/core/components/helpers) 里的 `<Grid>`。
