---
title: Rectangle and RegularPolygon
description: Use Rectangle and RegularPolygon for rectangles, rounded rectangles, and equal-sided polygons.
---

`<Rectangle>` fits cases where you know two corners, a center plus size, or one corner plus size. `<RegularPolygon>` is for equal-sided triangles, pentagons, hexagons, and similar shapes.

## Usage

```tsx
import { Rectangle, RegularPolygon } from '@retikz/react';
```

```tsx
<Rectangle corner1={[-40, -20]} corner2={[40, 20]} roundedCorners={6} />
<RegularPolygon center={[0, 0]} radius={36} sides={6} />
```

## Examples

### Rectangle

`<Rectangle>` can use two corners, or derive the corners from a center and size. Use `roundedCorners` for rounded rectangles.

| Form | Best for |
| --- | --- |
| `{ corner1, corner2 }` | Known opposite corners |
| `{ center, width, height }` | Known center and size |
| `{ center, side }` | Square |
| `{ corner1, width, height }` | Expand from one corner |

### RegularPolygon

`<RegularPolygon>` uses either a circumradius or a side length. `sides` must be at least `3`, and `rotate` changes the overall orientation.

| Form | Best for |
| --- | --- |
| `{ center, radius, sides }` | Known circumradius |
| `{ center, sideLength, sides }` | Known side length |

## How it works

> Optional for everyday use; revisit this section when a point form throws an error or the rotation direction looks surprising.

### Rectangle can pass through points

The `corner1 + corner2` form can pass points to the underlying Path, so it can reference node ids. Forms that require React-side size or vertex calculation, such as `center + width + height`, only accept literal `[x, y]` coordinates.

### RegularPolygon computes vertices first

`<RegularPolygon>` computes all vertices from the center, radius or side length, then expands to a closed Path. `rotate` changes the starting vertex angle; it is not an SVG transform.

## API Reference

| Component | Combination | Meaning |
| --- | --- | --- |
| `<Rectangle>` | `corner1` + `corner2` | Rectangle from two opposite corners |
| | `center` + `width` + `height` | Rectangle from center and size |
| | `center` + `side` | Square from center and side |
| | `corner1` + `width` + `height` | Rectangle from one corner and size |
| | + `roundedCorners?` | Rounded rectangle (clamped to half the smaller side) |
| `<RegularPolygon>` | `center` + `radius` + `sides` | Polygon from circumradius and side count |
| | `center` + `sideLength` + `sides` | Polygon from side length and side count |
| | + `rotate?` | Overall start angle |
