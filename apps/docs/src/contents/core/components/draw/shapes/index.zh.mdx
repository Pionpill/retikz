---
title: Shapes
description: 用 Circle、Rectangle、Grid 等形状组件快速画常见几何图形。
---

Shapes 是一组画几何图形的快捷组件。想画一个圆、一段弧、一个网格底纹或一颗星时，直接用对应组件；不用手写一长串 `<Path>` 和 `<Step>`。

如果图形里要放文字、要被连线吸附到边界上，用 [`<Node>`](/core/components/node/overview) 的 `shape`。Shapes 只负责“画出来”，不提供文本、锚点和可连接边界。

## 用法

```tsx
import { Circle, Rectangle, Grid } from '@retikz/react';
```

```tsx
<Circle center={[0, 0]} radius={32} fill="none" />
<Rectangle corner1={[-40, -20]} corner2={[40, 20]} roundedCorners={6} />
<Grid corner1={[-80, -40]} corner2={[80, 40]} step={20} zIndex={-1} />
```

### 先选哪一个

| 想画 | 用 |
| --- | --- |
| 圆、半圆、弓形 | `<Circle>` |
| 椭圆、部分椭圆 | `<Ellipse>` |
| 不闭合的弧线 | `<Arc>` |
| 扇形 | `<Sector>` |
| 矩形、圆角矩形 | `<Rectangle>` |
| 正多边形 | `<RegularPolygon>` |
| 星形 | `<Star>` |
| 背景网格、辅助线 | `<Grid>` |

这些组件都接收常见的 Path 视觉属性，例如 `stroke`、`strokeWidth`、`fill`、`dashPattern`、`opacity`、`arrow` 和 `zIndex`。

## 例子

### 圆

最常用的是 `center + radius`。需要半圆或弓形时，再加 `startAngle` / `endAngle`。

| 写法 | 适合 |
| --- | --- |
| `{ center, radius }` | 已知圆心和半径 |
| `{ center, diameter }` | 已知直径 |
| `{ from, to }` | 两点作为直径端点 |
| `{ corner1, corner2 }` | 放进一个包围盒 |

<ComponentPreview name="circle-basic" />

### 椭圆

椭圆和圆很像，只是半径分成 `radiusX` 和 `radiusY`。也可以用两个角给出一个包围盒。

| 写法 | 适合 |
| --- | --- |
| `{ center, radiusX, radiusY }` | 已知中心和两个半轴 |
| `{ center, diameterX, diameterY }` | 已知两个直径 |
| `{ corner1, corner2 }` | 放进一个包围盒 |

<ComponentPreview name="ellipse-basic" />

### 弧线

`<Arc>` 只画一段不闭合的弧。它适合做箭头弧线、环形标记或装饰线。

| 写法 | 适合 |
| --- | --- |
| `{ center, radius, startAngle, endAngle }` | 圆弧 |
| `{ center, radiusX, radiusY, startAngle, endAngle }` | 椭圆弧 |

角度从 +x 方向开始；`90` 指向屏幕下方。

<ComponentPreview name="arc-basic" />

### 扇形

`<Sector>` 会从圆心出发，画弧，再回到圆心。想要弦闭合的半圆或弓形，用 `<Circle>` 加角度。给 `innerRadius` 就画**空心扇形**（环形扇区 / donut 切片）；椭圆空心给 `innerRadiusX` + `innerRadiusY`。

| 写法 | 适合 |
| --- | --- |
| `{ center, radius, startAngle, endAngle }` | 圆扇形 |
| `{ center, radiusX, radiusY, startAngle, endAngle }` | 椭圆扇形 |
| 加 `innerRadius` | 空心圆扇形 |
| 加 `innerRadiusX` + `innerRadiusY` | 空心椭圆扇形 |

下图右侧三段用了不同的 `innerRadius`，体现空心半径如何控制环宽。

<ComponentPreview name="sector-basic" />

### 矩形

`<Rectangle>` 可以直接给两个角，也可以从中心和尺寸算出来。`roundedCorners` 用来画圆角。

| 写法 | 适合 |
| --- | --- |
| `{ corner1, corner2 }` | 已知两个对角 |
| `{ center, width, height }` | 已知中心和尺寸 |
| `{ center, side }` | 正方形 |
| `{ corner1, width, height }` | 从一个角向右下展开 |

<ComponentPreview name="rectangle-basic" />

### 正多边形

`<RegularPolygon>` 用外接半径或边长来确定大小。`sides` 至少为 3。

| 写法 | 适合 |
| --- | --- |
| `{ center, radius, sides }` | 已知外接半径 |
| `{ center, sideLength, sides }` | 已知边长 |

<ComponentPreview name="regular-polygon-basic" />

### 星形

`<Star>` 交替连接外半径和内半径上的点。`points` 是星角数量。

| 写法 | 适合 |
| --- | --- |
| `{ center, outerRadius, innerRadius, points }` | 自己控制内外半径 |
| `{ center, outerRadius, points, innerRatio? }` | 用比例自动算内半径 |

<ComponentPreview name="star-basic" />

### 网格

`<Grid>` 常用来做背景辅助线、坐标纸或分隔参考线。它比其它形状多一些控制项：可以分轴设置间距，只画横线或竖线，给主网格线单独设样式，也可以加外边框。

| 写法 | 适合 |
| --- | --- |
| `{ corner1, corner2 }` | 两个角确定范围 |
| `{ center, width, height }` | 中心和尺寸确定范围 |

`step` 同时控制横向和纵向间距；需要不同间距时用 `xStep` / `yStep`。

<ComponentPreview name="grid-basic" />

只需要一组参考线时，关掉另一个方向即可。

<ComponentPreview name="grid-lines" />

`majorEvery` 让每隔几条线变成主网格线，`border` 给网格加外边框。`origin` 可以让网格线按某个基准点对齐；`includeBoundary` 会在范围不能被步长整除时补上最后的边界线。

## 技术原理

> 日常使用可以跳过本节；只有在点位报错，或想把 Shapes 和 `<Path>` 混用时再回来读。

### 点位有两类

有些写法只是把点位交给底层 Path，可以引用节点 id、极坐标或普通 `[x, y]`。例如：

```tsx
<Circle center="A" radius={24} />
<Rectangle corner1="A" corner2="B" />
```

另一些写法需要组件先在 React 里算中点、宽高或顶点。这类写法只能用直接坐标 `[x, y]`：

```tsx
<Circle from={[-40, 0]} to={[40, 0]} />
<Grid corner1={[-80, -40]} corner2={[80, 40]} step={20} />
```

如果把节点 id 传给这类需要计算的写法，组件会报错，而不是猜一个结果。

### Shapes 仍然是 Path

Shapes 最后都会展开成等价的 `<Path>` / `<Step>`。这意味着：

- 它们可以和 Path 一样使用 `stroke`、`fill`、`arrow`、`zIndex`。
- 它们不会像 Node 那样产生可连接的锚点。
- 需要文本、锚点或边界吸附时，应该用 `<Node shape="circle">` 这类节点形状。

## API 参考

### 通用视觉属性

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `stroke` | `string` | `currentColor` | 描边颜色 |
| `strokeWidth` | `number` | `1` | 描边宽度 |
| `fill` | `string \| PaintSpec` | `none` | 填充：纯色或 PaintSpec（渐变 / 图案 / 图片） |
| `dashPattern` | `Array<number>` | — | 虚线段和间隔 |
| `opacity` | `number` | — | 整体透明度 |
| `arrow` | `'none' \| '->' \| '<-' \| '<->'` | — | 路径箭头 |
| `zIndex` | `number` | `0` | 同层栈序；数值越大越靠上 |

### 形状专属属性

每个组件按"属性组合"选定形态——含 `?` 的为可选附加属性。

| 组件 | 属性组合 | 含义 |
| --- | --- | --- |
| `<Circle>` | `center` + `radius` | 通过圆心和半径确定圆 |
| | `center` + `diameter` | 通过圆心和直径确定圆 |
| | `from` + `to` | 通过两点（直径两端）确定圆 |
| | `corner1` + `corner2` | 通过包围盒确定最大内接圆 |
| | + 角度（`startAngle`/`endAngle`/`sweepAngle` 三选二）+ `closed?` | 部分圆（`'chord'` 弦，默认 / `'open'` 纯弧） |
| `<Ellipse>` | `center` + `radiusX` + `radiusY` | 通过中心和两半轴确定椭圆 |
| | `center` + `diameterX` + `diameterY` | 通过中心和两直径确定椭圆 |
| | `corner1` + `corner2` | 通过包围盒确定内切椭圆 |
| | + 角度 + `closed?` | 部分椭圆（同 Circle） |
| `<Arc>` | `center` + `radius` + 角度 | 圆弧（不闭合；角度必给） |
| | `center` + `radiusX` + `radiusY` + 角度 | 椭圆弧 |
| `<Sector>` | `center` + `radius` + 角度 | 圆扇形（角度必给） |
| | `center` + `radiusX` + `radiusY` + 角度 | 椭圆扇形 |
| | + `innerRadius?`（椭圆用 `innerRadiusX?` + `innerRadiusY?`） | 空心扇形（环形扇区） |
| `<Rectangle>` | `corner1` + `corner2` | 通过两对角确定矩形 |
| | `center` + `width` + `height` | 通过中心和宽高确定矩形 |
| | `center` + `side` | 通过中心和边长确定正方形 |
| | `corner1` + `width` + `height` | 通过一角和宽高确定矩形 |
| | + `roundedCorners?` | 圆角矩形（clamp 到边长一半） |
| `<Grid>` | `corner1` + `corner2` | 通过两角确定网格范围 |
| | `center` + `width` + `height` | 通过中心和尺寸确定网格范围 |
| | `step`（或 `xStep` + `yStep`） | 步长：单值 / 分轴（分轴覆盖 `step`） |
| | `showVertical?` + `showHorizontal?` | 控制是否画竖线 / 横线 |
| | `origin?` + `offset?` | 控制网格对齐基准和相位偏移 |
| | `includeBoundary?` | 非整除范围内补最后边界线 |
| | `majorEvery?` + `majorOffset?` + `major*` | 每隔几条线使用主网格样式 |
| | `border?` + `borderPadding?` + `border*` | 显示外边框并单独设置边框样式 |
| | `borderRenderOrder?` + `clipToBorder?` | 控制边框绘制顺序和线段是否延伸到边框 |
| `<RegularPolygon>` | `center` + `radius` + `sides` | 通过外接半径和边数确定正多边形 |
| | `center` + `sideLength` + `sides` | 通过边长和边数确定正多边形 |
| | + `rotate?` | 整体起始角（缺省 −90°，一顶点朝上） |
| `<Star>` | `center` + `outerRadius` + `innerRadius` + `points` | 自定外/内半径和角数确定星形 |
| | `center` + `outerRadius` + `points` + `innerRatio?` | 按比例（缺省 0.5）算内半径 |
| | + `rotate?` | 整体起始角（缺省 −90°） |
