---
title: Draw Curves and Shapes
description: Use Draw way items for curve, cubic, bend, arc, circle, and ellipse.
---

Curve items replace the previous-to-next line with a curve. Shape items center on the previous item and do not consume the next item.

## Quadratic Bezier

`{ curve: [cx, cy] }` matches TikZ `..controls(B)..` and is equivalent to `<Step kind="curve">`.

<ComponentPreview name="draw-curve" />

## Cubic Bezier

`{ cubic: [[c1x, c1y], [c2x, c2y]] }` uses two control points to shape the start and end tangents, matching TikZ `..controls(B) and (C)..`.

<ComponentPreview name="draw-cubic" />

## Bend

`{ bend: 'left' | 'right', angle?: number }` gives only direction and angle; the compiler computes control points and lowers it to cubic.

<ComponentPreview name="draw-bend" size="lg" />

## Arc

`{ arc: { startAngle, endAngle, radius } }` draws an arc around the previous item and does not consume the next item.

<ComponentPreview name="draw-arc" />

## Circle

`{ circle: { radius } }` draws a full circle around the previous item; the pen returns to the center.

<ComponentPreview name="draw-circle" />

## Ellipse

`{ ellipse: { radiusX, radiusY } }` draws a full ellipse centered on the previous item.

<ComponentPreview name="draw-ellipse" />
