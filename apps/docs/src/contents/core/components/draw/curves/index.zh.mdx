---
title: Draw 曲线与形状
description: 用 Draw way 写 curve、cubic、bend、arc、circle 与 ellipse。
---

曲线类 item 会把上一项到下一项的直线替换为曲线；形状类 item 以上一项为圆心或中心，不消耗下一项。

## 二次贝塞尔

`{ curve: [cx, cy] }` 对应 TikZ `..controls(B)..`，等价 `<Step kind="curve">`。

<ComponentPreview name="draw-curve" />

## 三次贝塞尔

`{ cubic: [[c1x, c1y], [c2x, c2y]] }` 用两个控制点分别影响起点与终点切线，对应 TikZ `..controls(B) and (C)..`。

<ComponentPreview name="draw-cubic" />

## Bend

`{ bend: 'left' | 'right', angle?: number }` 只给方向与角度，编译期自动算控制点并降为 cubic。

<ComponentPreview name="draw-bend" size="lg" />

## 弧段

`{ arc: { startAngle, endAngle, radius } }` 以上一项为圆心画弧段，不消耗下一项。

<ComponentPreview name="draw-arc" />

## 整圆

`{ circle: { radius } }` 以上一项为圆心画整圆，画完画笔回到圆心。

<ComponentPreview name="draw-circle" />

## 整椭圆

`{ ellipse: { radiusX, radiusY } }` 以上一项为中心画整椭圆。

<ComponentPreview name="draw-ellipse" />
