---
title: Draw Edge Labels
description: Use the Draw way label operator to attach an edge label to the next path segment.
---

`{ label }` is an infix label operator in Draw way. It decorates the next produced segment by setting IR `step.label`; it does not produce a segment itself.

## Basic Form

```tsx
<Draw way={['a', { label: 'ok' }, 'b']} />
```

The shorthand `{ label: 'ok' }` is equivalent to `{ label: { text: 'ok' } }`. The object form can also set `position` and `side`.

<ComponentPreview name="draw-label" size="lg" />

## Position Rules

`label.position` accepts seven keywords or a numeric `0..1` t. The geometry of t depends on step kind: normalized arc length for line, equal slices for fold, Bezier parameter for curves, and angular parameterization for arc / circle / ellipse.

For the full Kernel behavior, see [Step edge labels](/core/components/draw/step#edge-label-label).
