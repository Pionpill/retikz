`<Path>` 是底层路径容器：children 必须是若干 `<Step>`（至少 2 个），定义路径的 move / line / step（折角）/ cycle（闭合）等动作序列。日常连线用 `<Draw>` 更简洁，需要折角、闭合、自定义 stroke、不同 step kind 时回到 `<Path>`。

## 用法

```tsx
import { Path, Step } from '@retikz/react';
```

```tsx
<Path stroke="currentColor">
  <Step kind="move" to="a" />
  <Step kind="line" to="b" />
</Path>
```

## 组合

`<Path>` 必须搭配 `<Step>` 子节点（至少 2 个）；首个非 `move` 时编译器会强制改成 `move`：

```tsx
<Path>
  <Step kind="move" to={...} />     {/* 第 1 个：必为 move */}
  <Step kind="line" to={...} />     {/* 之后任意多个 line / move / step */}
  ...
</Path>
```

`<Step>` 支持 `move` / `line` / `step` / `cycle` 四种 kind——前两种画直线段，`step` 走直角折角，`cycle` 闭合回起点，详见 [Step](/core/components/draw/step)。

### 箭头

`arrow` 控制方向（4 个枚举：`'->'` 终点、`'<-'` 起点、`'<->'` 两端、`'none'` / 省略 = 无），`arrowShape` 控制形状（5 个枚举，默认 `'normal'`）：

| 形状 | 描述 | 典型用途 |
| --- | --- | --- |
| `'normal'` | 实心三角 | 流程图默认 |
| `'open'` | 空心三角 | UML 泛化 / 继承 |
| `'stealth'` | 倒钩三角 | 现代流图、视觉锐利 |
| `'diamond'` | 实心菱形 | UML 组合 |
| `'openDiamond'` | 空心菱形 | UML 聚合 |
| `'circle'` | 实心圆点 | 连接点 / dot end |
| `'openCircle'` | 空心圆点 | 白圈端 / o-end |

<ComponentPreview name="path-arrow" />

> 多节点路径（如 A → B → C）按段独立 clip，但箭头**只**贴在整体起点 / 终点——中间节点不会出现额外箭头。颜色由 marker 的 `context-stroke` 自动同 stroke；大小按 `strokeWidth` 缩放。

## API 参考

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `stroke` | `string` | `currentColor` | 描边色 |
| `strokeWidth` | `number` | `1` | 描边宽度 |
| `strokeDasharray` | `string` | — | SVG `stroke-dasharray` 模式 |
| `arrow` | `'none' \| '->' \| '<-' \| '<->'` | — | 路径级箭头方向 |
| `arrowShape` | `'normal' \| 'open' \| 'stealth' \| 'diamond' \| 'openDiamond' \| 'circle' \| 'openCircle'` | `'normal'` | 箭头形状 |
| `children` | `Array<<Step>>` | — | 至少 2 个；首个非 `move` 时编译器会强制改成 `move`。`<Step>` 支持 `move` / `line` / `step`（折角）/ `cycle`（闭合） |

## 与 Draw 的取舍

| | `<Draw>` | `<Path>` + `<Step>` |
| --- | --- | --- |
| 写法 | 一行 way 数组 | 容器 + 子节点 |
| 表达力 | 与 Path 等价（编译展开） | 全部 Kernel 能力 |
| 适用 | 90% 连线场景 | 显式 Step 序列、未来扩展 |

`<Draw>` 由 `<Path>` 编译而成；写文档 / 教程时一律先讲 `<Draw>`，`<Path>` 是给"想看底层"的用户。
