title: Arc
description: Shape sugar — draw a circular / elliptical arc (open, unclosed) in one line; expands to Path + arc step.
---

`<Arc>` draws an **open** arc: give a center + radius + angles. Circular (`radius`) or elliptical (`radiusX` / `radiusY`). It expands to `<Path>` + `<Step kind="arc">` (with an explicit `center`).

Arc **requires angles** — that is what distinguishes it from the angled forms of [Circle](/core/components/circle) / [Ellipse](/core/components/ellipse): Arc stays open (a pure arc), while Circle with angles defaults to `chord` closing. For a wedge closed through the center use [Sector](/core/components/sector).

## Usage

```tsx
import { Arc } from '@retikz/react';
```

```tsx
<Arc center={[0, 0]} radius={40} startAngle={0} endAngle={90} />
```

## Forms

| Form | prop |
| --- | --- |
| circular arc | `{ center, radius, ...angles }` |
| elliptical arc | `{ center, radiusX, radiusY, ...angles }` |

`center` is pass-through (any Target) — Arc computes nothing in the component, so the center can be a node id / polar point. Angles: any two of `startAngle` / `endAngle` / `sweepAngle`.

## API reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `center` | `Target` | — | Center (any Target) |
| `radius` | `number` | — | Circular arc radius (xor radiusX/radiusY) |
| `radiusX` / `radiusY` | `number` | — | Elliptical arc semi-axes |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | Angles (any two, required) |
| Visual props | same as `<Path>` | — | `stroke` / `strokeWidth` / `arrow` / `zIndex` etc. |
