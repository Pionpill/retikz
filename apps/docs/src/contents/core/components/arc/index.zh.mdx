`<Arc>` 画一段**不闭合**的弧线：给圆心 + 半径 + 角度。圆弧（`radius`）或椭圆弧（`radiusX` / `radiusY`）。它编译期展开为 `<Path>` + `<Step kind="arc">`（用显式 `center`）。

Arc **必给角度**——这是它与 [Circle](/core/components/circle) / [Ellipse](/core/components/ellipse) 带角度形态的区别：Arc 不闭合（纯弧线），Circle 带角度默认 `chord` 闭合。要画经圆心闭合的扇形用 [Sector](/core/components/sector)。

## 用法

```tsx
import { Arc } from '@retikz/react';
```

```tsx
<Arc center={[0, 0]} radius={40} startAngle={0} endAngle={90} />
```

## 形态

| 形态 | prop |
| --- | --- |
| 圆弧 | `{ center, radius, ...角度 }` |
| 椭圆弧 | `{ center, radiusX, radiusY, ...角度 }` |

`center` 透传（任意 Target）——Arc 不在组件内算坐标，圆心可用 node id / 极坐标。角度三键 `startAngle` / `endAngle` / `sweepAngle` 三选二。

## API 参考

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `center` | `Target` | — | 圆心（任意 Target） |
| `radius` | `number` | — | 圆弧半径（与 radiusX/radiusY 二选一） |
| `radiusX` / `radiusY` | `number` | — | 椭圆弧两半轴 |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | 角度（三选二，必给） |
| 视觉 prop | 同 `<Path>` | — | `stroke` / `strokeWidth` / `arrow` / `zIndex` 等 |
