title: Sector

`<Sector>` draws a wedge (pie slice): give a center + radius + angles for a wedge closed through the center. Circular (`radius`) or elliptical (`radiusX` / `radiusY`).

It expands to a `<Path>`: `move(arcStart) → arc(explicit center) → line(center) → cycle` — the explicit `center` keeps the arc centered correctly, and `cycle` closes it cleanly into a wedge. `center` must be literal Cartesian (the component computes the arc start).

## Usage

```tsx
import { Sector } from '@retikz/react';
```

```tsx
<Sector center={[0, 0]} radius={40} startAngle={0} endAngle={90} fill="#eee" />
```

## Forms

| Form | prop |
| --- | --- |
| circular sector | `{ center, radius, ...angles }` |
| elliptical sector | `{ center, radiusX, radiusY, ...angles }` |

`center` must be **literal `[x, y]`** (the component computes the arc start). Angles: any two of `startAngle` / `endAngle` / `sweepAngle`, required.

For chord closing (not through the center) use [Circle](/core/components/circle) with angles (chord is the default).

## API reference

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `center` | `[x, y]` | — | Center (literal) |
| `radius` | `number` | — | Circular sector radius (xor radiusX/radiusY) |
| `radiusX` / `radiusY` | `number` | — | Elliptical sector semi-axes |
| `startAngle` / `endAngle` / `sweepAngle` | `number` | — | Angles (any two, required) |
| Visual props | same as `<Path>` | — | `fill` / `stroke` / `fillOpacity` / `zIndex` etc. |
