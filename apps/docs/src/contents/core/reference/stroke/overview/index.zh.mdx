---
title: Stroke 概述
description: 笔画系统 —— Path / Step / Target
---

`stroke` 涵盖构成一条画线的所有 IR 原件：

- [`Path`](/core/reference/stroke/path) —— 外壳容器，承载描边样式、箭头形状、填充，以及 `children` step 序列
- [`Step`](/core/reference/stroke/step) —— path 上一个动作（按 `kind` 判别 10 种：move / line / fold / cycle / curve / cubic / bend / arc / circlePath / ellipsePath）
- [`Target`](/core/reference/stroke/target) —— step 的目标点（笛卡尔 / 极 / node id / 相对偏移）
