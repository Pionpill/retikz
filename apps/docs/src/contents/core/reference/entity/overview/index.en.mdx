---
title: Entity
description: Named members placed in a scene — Node and Coordinate
---

`entity` covers IR primitives that **have an id**, are **placed at a position**, and **don't draw paths themselves**. Two entities exist today:

- [`Node`](/core/reference/entity/node) — positioned shape (rectangle / circle / ellipse / diamond) with optional text content
- [`Coordinate`](/core/reference/entity/coordinate) — invisible named anchor; usable as a path target or a position-of reference

Both register into the same node index, so a `Coordinate` id and a `Node` id share the same namespace.
