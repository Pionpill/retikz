This group is the runtime pipeline from IR to picture. `compile` is the entry point: it takes IR and emits a renderer-agnostic Scene Primitive; the Primitive is that renderer-agnostic drawing model; the Parser is the set of pure functions that turn sugar input into IR fragments. Look here when you drive compilation yourself, write a custom renderer, or reuse the sugar parsing logic.

> This is the reference layer, not component docs; to author figures, start from [Components](/core/components/layout).

## What each part covers

| Topic | Responsibility | Reach for it when |
| --- | --- | --- |
| Compile | The IR → Scene entry point, layout options, text measuring, warning contract | Driving compilation yourself, configuring layout, receiving warnings |
| Primitive | The renderer-agnostic drawing model produced by compile | Writing a custom renderer / exporter |
| Parser | Pure conversion from sugar input to IR fragments | Reusing sugar parsing such as relative coordinates |
