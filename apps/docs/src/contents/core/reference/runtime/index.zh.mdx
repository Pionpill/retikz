---

这一分组是从 IR 到画面的运行时管线。`compile` 是入口，吃 IR、吐出渲染目标无关的 Scene Primitive；Primitive 是那份与具体渲染器无关的绘制模型；Parser 是把 Sugar 输入转成 IR 片段的纯函数。自己驱动编译、写自定义渲染器、或复用 Sugar 解析逻辑时看这里。

> 这里是参考层，不是组件文档；写图请从 [组件](/core/components/layout) 开始。

## 职责一览

| 主题 | 职责 | 何时查阅 |
| --- | --- | --- |
| Compile | IR → Scene 的编译入口、布局选项、文字度量、warning 合同 | 自己驱动编译、配置布局、接收 warning |
| Primitive | compile 输出的渲染目标无关绘制模型 | 写自定义渲染器 / 导出器 |
| Parser | Sugar 输入到 IR 片段的纯函数转换 | 复用相对坐标等 Sugar 解析逻辑 |
