A `position` field can be filled with any of three forms. All resolve to Cartesian at compile time.

## Position

`PositionSchema` is a tuple — the simplest position form.

`PolarPositionSchema` is recursive — the `origin` field may itself be another `PolarPosition` (chained polars).
