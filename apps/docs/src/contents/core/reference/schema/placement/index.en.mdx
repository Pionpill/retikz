description: Four forms a `position` field can take — Cartesian / Polar / At-relative / Offset
---

A `position` field can be filled with any of four forms. All resolve to Cartesian at compile time.

## Position

`PositionSchema` is a tuple — the simplest position form.

<ZodSchema name="PositionSchema" />

## PolarPosition

`PolarPositionSchema` is recursive — the `origin` field may itself be another `PolarPosition` (chained polars).

<ZodSchema name="PolarPositionSchema" />

## AtPosition

`AtPositionSchema` mirrors TikZ `[<direction>=<distance> of <id>]` from the positioning library.

<ZodSchema name="AtPositionSchema" />

## OffsetPosition

`OffsetPositionSchema` mirrors TikZ `calc` library syntax `($(of) + (dx, dy)$)` — an arbitrary `(dx, dy)` offset from a base point; the `of` union shape matches `PolarPosition.origin`.
