`@retikz/math` 是 retikz 的**零依赖纯计算几何**包：向量运算、仿射变换、arc 原语、求交、三角形内外接圆、点在多边形、凸包。全是纯函数 + 普通 `Position = [number, number]` 数组，不写 class、零 IR、零 zod、零运行时依赖。

`@retikz/core` 依赖它做底层计算。常用原语（`Position` / `point` / `lerp` / `localToWorld` / `worldToLocal`）**已由 `@retikz/core` re-export**——日常画图直接用 core 即可，不必单独安装 math。需要 core 没有转出的能力（求交、内外接圆、凸包等）时，再直接依赖 `@retikz/math`。

> 这里是参考层，不是组件文档；写图请从 [组件](/core/components/layout) 开始。

## 安装

```bash
pnpm add @retikz/math
```
