---
title: AtPosition
description: 相对定位 —— 在被引用节点的某方向以某距离
---

`AtPositionSchema` 对应 TikZ positioning library 的 `[<direction>=<distance> of <id>]`。

<ZodSchema name="AtPositionSchema" descriptions={{
  direction: '从被引用节点指向本节点的方向，视觉约定（above = 视觉向上，screen y-）',
  of: '被引用 node / coordinate 的 id；必须在 IR 中先于本节点出现（不接受前向引用）',
  distance: '从被引用节点中心到本节点中心的距离（用户单位）；默认走 Tikz nodeDistance 上下文，再回 1',
}} />
