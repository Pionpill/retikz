`<BarMark>` 把每行数据画成一根柱：`x` 绑**分类字段**（`<Plot>` 自动用 band 比例尺，把每个类别分到等宽的一段），`y` 绑数值字段，柱从基线 `0` 画到该值。

## 多系列：分组与堆叠

给 `series` 指定一个字段，数据就按它拆成多个系列，每个系列自动取一种颜色。柱默认**并排**（dodge）——同一类别下的各系列挤在一格里横向排开：

<ComponentPreview name="bar-grouped" sourceFiles={['bar-grouped.data.ts']} />

<p className="text-sm text-muted-foreground">同一季度的 A / B 两系列并排。</p>

加上 `stack`，多系列改为**累叠**——后一个系列从前一个的顶端接着往上画。堆叠需要的累积区间由 `<Plot>` 自动算好，你只声明 `stack`：

<ComponentPreview name="bar-stacked" sourceFiles={['bar-stacked.data.ts']} />

<p className="text-sm text-muted-foreground">同一季度的 A / B 两系列上下累叠，柱顶是合计。</p>

## 极坐标：径向柱

把 `<Plot coordinate="polar2D">` 一开，同一个 `<BarMark>` 自动变成径向柱——角向是类别（自动 band），径向是值，柱从圆心向外长。不需要换组件，也不用改字段绑定。

<ComponentPreview name="bar-radial" sourceFiles={['bar-radial.data.ts']} />

<p className="text-sm text-muted-foreground">同一份 BarMark，仅加 <code>coordinate="polar2D"</code> 即成径向柱。</p>

## API

| prop | 类型 | 说明 |
| --- | --- | --- |
| `x` | `string` | 绑 x 通道的字段路径（分类，自动 band 比例尺） |
| `y` | `string` | 绑 y 通道的字段路径（数值，决定柱高） |
| `series` | `string?` | 系列字段：按它拆多系列；省略为单系列 |
| `stack` | `boolean?` | 多系列时改为堆叠；省略时多系列并排（dodge） |
| `color` | `string?` | 颜色字段；省略时取 `series` |
| `id` | `string?` | 可选句柄（预留，暂不解析） |
