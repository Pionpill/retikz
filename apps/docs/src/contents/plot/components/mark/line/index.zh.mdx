---
title: 折线
description: 用 <LineMark> 画折线图：按 order 连点，支持多系列；polar 下加 closed 即雷达。
---

`<LineMark>` 把数据点按顺序连成一条折线。`x` / `y` 绑位置字段，`order` 决定连线顺序（缺省按数据数组顺序）。给 `series` 拆成多条线，给 `closed` 首尾相接闭合——在 polar 下就是雷达。

## 用法

```tsx
import { Plot, LineMark } from '@retikz/plot-react';
```

```tsx
<Plot data={revenue}>
  <LineMark x="month" y="revenue" order="month" />
</Plot>
```

`order` 推荐显式写——否则折线按数据行顺序连，数据顺序一变折线就乱跳。

## API

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `x` | `string` | — | 绑 x 通道的字段路径（polar 下作角向） |
| `y` | `string` | — | 绑 y 通道的字段路径（polar 下作径向） |
| `angle` | `string?` | `x` | polar 下显式绑角向字段；缺省复用 `x` |
| `radius` | `string?` | `y` | polar 下显式绑径向字段；缺省复用 `y` |
| `order` | `string?` | 数据顺序 | 驱动连接顺序的字段 |
| `series` | `string?` | — | 系列字段：拆成多条折线；缺省单线 |
| `closed` | `boolean?` | `false` | 末点回连首点闭合成多边形（polar 下即雷达轮廓） |
| `color` | `string?` | `series` | 颜色字段（自动 ordinal 色 scale）；缺省取 `series` |
| `id` | `string?` | — | 可选句柄（预留，暂不解析） |
