title: 面积

---

`<AreaMark>` 在折线的基础上多了一条回边：上沿是 `x` / `y` 连成的折线，下沿贴附 `baseline`（缺省 `0`），两者围出一块可填充的区域。切到 polar 坐标系并加 `closed`，区域闭合成多边形，就是填充版雷达。

## 用法

```tsx
import { Plot, AreaMark } from '@retikz/plot-react';
```

```tsx
<Plot data={activity}>
  <AreaMark x="day" y="users" order="day" />
</Plot>
```

`x` / `y` 绑位置字段，`order` 决定上沿的连接顺序。默认下沿是 `0`，需要别的基线时给 `baseline`。

## 例子

### 基础面积

上沿折线与基线 `0` 围成填充区域，适合强调累计量或趋势的“体量”。

<ComponentPreview name="area-basic" sourceFiles={['area-basic.data.ts']} />

### 填充雷达

`<Plot coordinate="polar">` 加 `<AreaMark closed />`：末点回连首点闭合成多边形，并带填充——比 `<LineMark closed />` 的线框雷达多了填充面。各维度沿角向均布，值映射到半径。

<p className="text-sm text-muted-foreground">closed 让首尾相接成闭合多边形；polar 下即填充雷达。</p>

## API

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `x` | `string` | — | 绑 x 通道的字段路径（polar 下作角向） |
| `y` | `string` | — | 绑 y 通道的字段路径（polar 下作径向） |
| `angle` | `string?` | `x` | polar 下显式绑角向字段；缺省复用 `x` |
| `radius` | `string?` | `y` | polar 下显式绑径向字段；缺省复用 `y` |
| `order` | `string?` | 数据顺序 | 驱动上沿连接顺序的字段 |
| `baseline` | `number?` | `0` | 回边贴附的基线值 |
| `closed` | `boolean?` | `false` | 末点回连首点闭合成多边形（polar 下即填充雷达） |
| `series` | `string?` | — | 系列字段：拆成多块面积；缺省单块 |
| `color` | `string?` | `series` | 颜色字段（自动 ordinal 色 scale）；缺省取 `series` |
| `id` | `string?` | — | 可选句柄（预留，暂不解析） |
