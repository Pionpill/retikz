`<PointMark>` draws one dot per data row. Bind `x` / `y` to position fields. Give `size` a numeric field and the dot radius maps to the value as a bubble chart; give `color` a categorical field and dots are tinted by category.

## Usage

```tsx
import { Plot, PointMark } from '@retikz/plot-react';
```

```tsx
<Plot data={cities}>
  <PointMark x="lng" y="lat" />
</Plot>
```

Scatter often overlays a line — draw `<LineMark>` first, then layer `<PointMark>` on the same data to mark each observation.

## API

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `string` | — | Field path bound to the x channel (angular under polar) |
| `y` | `string` | — | Field path bound to the y channel (radial under polar) |
| `size` | `string?` | — | Numeric field: mapped to dot radius via a square-root scale (area-perceptual); negative values error |
| `opacity` | `string?` | — | Continuous field: mapped to opacity via a clamped linear scale to `[min, 1]`; temporal / categorical fields error |
| `color` | `string?` | — | Color field (categorical, auto ordinal color scale); colors per datum. Continuous / temporal fields error |
| `id` | `string?` | — | Optional handle (reserved, not yet resolved) |
