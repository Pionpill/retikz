## 安装

React 项目里使用 plot 时，入口包是 `@retikz/plot-react`。它会带上 `@retikz/plot` 与底层 retikz 依赖；你的应用仍需要提供 React peer 依赖。

## 一步步搭建

### 1. 准备裸数据行

数据是普通对象数组，不进 IR，也不需要包成特殊类。字段名会在 mark 里通过 `x` / `y` props 引用。

```ts
const sales = [
  { month: 0, revenue: 10 },
  { month: 1, revenue: 14 },
  { month: 2, revenue: 9 },
  { month: 3, revenue: 18 },
  { month: 4, revenue: 15 },
];
```

### 2. 用 `<Plot>` 和 `<LineMark>` 画趋势

`<Plot>` 接收数据和显示尺寸；`<LineMark>` 声明哪个字段放到 x / y 通道。`order="month"` 表示按月份连线，避免数据行顺序变化时折线乱跳。

`<Plot>` 是薄容器、自己不画轴：要看到坐标轴，得显式加 `<Axis>`——`dimension` 选 x / y 维度，`grid` 让该维度带网格。下面 y 轴带网格、x 轴不带。

```tsx
<Plot data={sales} width={360} height={200}>
  <LineMark x="month" y="revenue" order="month" />
  <Axis dimension="x" />
  <Axis dimension="y" grid />
</Plot>
```

### 3. 叠一层 `<PointMark>`

同一张 `<Plot>` 里的 mark 共用坐标系和比例尺。再加一个 `<PointMark>`，就能在折线上标出每个观测点。

```tsx
<Plot data={sales} width={360} height={200}>
  <LineMark x="month" y="revenue" order="month" />
  <PointMark x="month" y="revenue" />
  <Axis dimension="x" />
  <Axis dimension="y" grid />
</Plot>
```

到这里先记住 4 个规则：

- `data` 是裸数据行数组，字段值必须能转成有限数字。
- `<LineMark>` / `<PointMark>` 只声明字段映射，不直接写像素坐标。
- `<Plot>` 是薄容器、自己不画轴——上面的 `<Axis dimension="x" />` 和 `<Axis dimension="y" grid />` 才显式声明了 x/y 轴和 y 网格。
- 代码面板里的 IR tab 展示的是 `<Plot>` 装配出的 Plot IR；数据仍然单独传入。

## 常用开关

`<Plot>` 只画你列出的 `<Axis>`；`<Axis>` 上可调刻度数 / 标签 / 网格：

```tsx
<Plot data={sales} width={360} height={200}>
  <LineMark x="month" y="revenue" order="month" />
  <Axis dimension="y" grid tickCount={5} />
</Plot>
```

只想画数据、不画轴和网格时，用 `bare`：

```tsx
<Plot data={sales} width={360} height={200} bare>
  <LineMark x="month" y="revenue" order="month" />
</Plot>
```

`width × height` 是包含坐标轴和刻度标签在内的外框。刻度标签太挤时，用 `fontSize` 调字号，用 `margin` 覆盖四周留白：

```tsx
<Plot data={sales} width={360} height={200} fontSize={12} margin={{ left: 48 }}>
  <LineMark x="month" y="revenue" order="month" />
</Plot>
```

## 时间轴

x 是日期时，给 `<Plot scaleX="time">`——刻度会落在整月 / 整年这样的时间边界上，标签也按时间格式化。日期字段可以是 ISO 字符串或毫秒时间戳。
